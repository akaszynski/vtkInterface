    Working with conda?
    ^^^^^^^^^^^^^^^^^^^

    PyVista is available on `conda-forge <https://anaconda.org/conda-forge/pyvista>`_

    ++++++++++++++++++++++

    .. code-block:: bash

        conda install -c conda-forge pyvista

    ---

    Prefer pip?
    ^^^^^^^^^^^

    PyVista can be installed via pip from `PyPI <https://pypi.org/project/pyvista>`__.

    ++++

    .. code-block:: bash

        pip install pyvista
